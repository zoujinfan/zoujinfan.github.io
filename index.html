Redis介绍

本人在运维项目中担任OA角色，运维过程中涉及Redis数据库知识。本文主要对书籍《Redis实战》进行大致了解，对OA需了解的知识点进行梳理整理，主要总结基本概念，实战部分咱不涉及。

一、初识Redis

  • Redis简单介绍

相对于普通数据库（mysql）数据存在硬盘，Redis是存在内存中。

-> 特点：访问速度快

-> 适用场景：存临时的、需快速访问的数据。例如：游戏排行榜、视频播放量（计数器）、临时验证码（会话存储）

  • Azure Cache for Redis

Azure云平台提供了Redis服务，称“Azure Cache for Redis”

图片解释：

1. 有缓存命中（Cache Hit） ✅

  • request → Azure Redis
  • Redis 里有数据 → 直接 response 给请求方

2. 缓存未命中（Cache Miss，数据库有数据） ✅

  • request → Azure Redis
  • Redis 没有数据 → 由你的应用去访问数据库
  • 数据库返回数据 → 你的应用把数据写回 Redis 做缓存，然后再返回给用户

注意：Redis 不会主动去查数据库，它只负责存和取。去数据库拿数据的逻辑是你代码里写的。

3. 缓存未命中（数据库也没数据） ✅

  • request → Azure Redis → Redis 没有
  • 应用去数据库 → 数据库也没有
  • 应用返回空给用户（通常不会往 Redis 里写空值，除非做“空缓存”防穿透）。

二、Redis特性（数据安全与性能保障）

复制

原文表达：

需要扩展读请求，或者在需要写入临时数据的时候，用户可以通过设置额外的Redis从服务器来保存数据集的副本。

在接收到主服务器发送的数据初始副本（initial copy of the data）之后，客户端每次向主服务器进行写入时，从服务器都会实时地得到更新。

在部署好主从服务器之后，客户端就可以向任意一个从服务器发送读请求了，而不必再像之前一样，总是把每个请求都发送给主服务器（客户端通常会随机地选择适用哪个从服务器，从而将负载平均分配到各个从服务器上）。

图中说明：

RDB 在 Redis 里是 Redis DataBase file 的缩写，指的是 快照持久化（Snapshotting） 生成的文件。

从服务器在进行同步时，会清空自己的所有数据

  • 主从复制（Master-Slave Replication）

    主服务器的数据（master） 完整拷贝到 从服务器（slave）上，并且保持同步。（第一次连接时，全量复制；后续再有更新，立马主同步到从上）

    优点：

    1. 读写分离：主机负责写，从机负责读

    2. 高可用：主机挂了，从机顶

    3. 数据备份：防止单机数据丢失

    • 主从链

      原文表达：随着负载不断上升，主服务器可能会无法快速地更新所有从服务器，或者因为重新连接或者重新同步从服务器而导致系统超载。为了缓解这个问题，用户可以创建一个由Redis主从节点（master/slave node）组成的中间层来分担主服务器的复制工作。

      选择主从链的主要场景：

持久化（将数据存储到硬盘）

  • RDB（Redis DataBase file, 快照snapshotting持久化生成的文件，某一时刻的所有数据写入硬盘），默认文件名dump.rdb

  • AOF（append-only file, 只追加文件，在执行写命令时将被执行的写命令复制到硬盘），Redis会把每条写命令记录在日志文件（appendonly.aof）里

三、处理系统故障

  1. 验证持久化文件是否完好无损（无论是快照持久化还是AOF持久化），并在有需时对其修复

  2. 更换故障主服务器

    方法 简单介绍
    Redis Sentinel 当主节点故障时，Sentinel 自动检测并将某个从节点提升为新的主节点（自动故障恢复）
    Redis Cluster Redis Cluster 中每个主节点都有对应的从节点，主节点宕机时从节点自动接管（自动分片和故障转移，需要客户端支持）
    手动切换 确认主节点故障，停止主节点服务，从节点提升主节点（小规模或临时应急）
    热备切换 准备好一台新的 Redis 服务器，将新的 Redis 配置为从节点，同步数据，等待数据同步完成后，将该节点提升为主节点（手动或自动），关闭旧主节点，业务端切换到新主节点（需要平滑切换的场景）

四、其他知识点

  1. 客户端分片（client-side sharding）:指由客户端负责将数据分布到多个 Redis 实例上，而不是由 Redis 服务器自身来管理数据分片。客户端根据一定的规则（如哈希算法）决定将某个 key 存储到哪个 Redis 实例。缺乏自动故障转移，某个 Redis 节点宕机，客户端需手动处理或重新映射分片。

  2. 键的过期时间：通过Redis的过期时间（expiration）特性来让一个键在给定的时限（timeout）之后自动被删除。

五、运维真实案例

问题：微软host os升级导致Redis（缓存服务器）故障转移（failover）

采取的措施：

  1. Redis CPU 高时采取重启策略，属于快速恢复服务可用性的“应急手段”，能短时间解除阻塞和异常状态。然后再重启业务服务，以重建与 Redis 的连接并刷新缓存，确保两者状态同步，保证业务稳定运行。

  2. 增加Redis服务副本（replica，也称“从节点”或“副本节点”），主要体现在高可用性、读性能提升，故障恢复能力增强等方面。

  3. HA（High Availabilty, 高可用性），是保障服务“持续在线”的核心能力，依靠主从复制、自动故障转移、负载均衡等机制，在故障时自动或快速恢复服务，避免影响用户体验。

其他说明：

Redis是运行在主机内存中的进程，当宿主机操作系统（Host Operation System）升级时，通常升级伴随内核升级、系统服务更新，很多升级操作必须重启才能生效，宿主机重启 redis进程就会停止。

怀念-爷爷邹学保

​ 我从不写东西，也不习惯文字记录生活。可爷爷走后，我想把他的点滴写下来。哪怕平凡，也值得被记住。
​ 因为在这个世界上，有一个人叫邹学保，曾认真地活过。
​ 死亡不是终点，遗忘才是。
​ 而我们，一直记得您。

爷爷的前半辈子

​ 爷爷生于农历1930年腊月初二。我2000年上幼儿园时，爷爷已经年过七十，脸上满是岁月的皱纹。对我来说，从有记忆起，爷爷就已经是个慈祥的老人了，我也从未见过他年轻时的模样，更别说照片。

​ 但关于爷爷年轻时的点滴，我还是听过一些。

​ 爷爷出生在战火纷飞的抗日年代。他曾对我说，小时候有一次被日本鬼子背着在山上跑。那时我听得目瞪口呆，难以置信地跑去问妈妈，妈妈笑着说爷爷又在吹牛了。爷爷也只是呵呵一笑，没再多说。这件事或许只是爷爷随口一提，但对年幼的我来说，却是一段刻骨铭心的记忆。

​ 爷爷小时候生活的村子靠山，山上有豺狼出没。白天它们也会跑进村子，有时小孩或大人一不留神就会被叼走。爷爷小时候就在院子里被豺狼咬住大腿，幸好村里人及时赶来，将豺狼赶跑，爷爷才得以幸免。他的腿上一直留着那道四五厘米长的弯弯牙印。夏天爷爷穿短裤，我总爱盯着那印子看，心里总是忍不住感叹：爷爷真牛，大难不死，必有后福。

​ 爷爷记不起自己的亲生父母。他说自己很小的时候就被遗弃，是村里人收养长大的。村里大多是一个姓的族人，都姓“邹”，所以叫邹家村。爷爷就取名叫“邹学保”。
​ “学保”这个名字，听起来像是希望他好好读书、平安长大。但爷爷没有上过学，他常笑说：“那时候，能活下去就不错了。”不过爷爷算术很好，偶尔还会拿书问我不认识的字怎么读。“保”——保护自己、保佑健康。的确，爷爷一生很少生病，只是年纪大后摔断过几次骨头。想想看，这名字还真挺不错。

​ 成年后的爷爷曾当过村干部，做事踏实，事事为村里人着想，深受村里人的尊敬爱戴。成家后，他前后育有四个女儿，直到四十岁那年才得一子，也就是我的爸爸。老来得子，自然格外疼爱。可惜，爷爷五十岁左右，奶奶就因病去世。从那以后，爷爷一个人把孩子们拉扯大，终身未再娶。

​ 爷爷对每个孩子都很好。女儿，儿子，女婿，媳妇，孙子孙女也都对爷爷很好，重孙们也都喜欢围着他转。村里人常说，爷爷晚年很享福。

​ 我和姐姐、爸妈、爷爷五口人住在一起。小时候，每天吃饭前我都会认真地数好五副碗筷。妈妈说，他们一家人最初住在石瓦房里，夏天常有蜈蚣爬进屋咬人，爸爸被咬了好几次。后来咬咬牙借钱，终于建起了两层楼的新房。等我出生时，已经搬进了新房。

​ 我和爷爷的记忆，也都发生在那栋两层的小楼里。

​ 如今，那座楼也已经成为记忆中的“老房子”。建于1995年，拆于2020年。它见证了四代人的生活，承载了太多回忆，如今也只能留存在脑海里。

老房子

和爷爷在老房子的回忆

​ 老房子是个两层的小楼，外观很像扑克牌K人像的样子。宽敞的院子，三个厢房，其中一个是厨房，一个接待客人吃饭的厢房，另一个房间堆放杂物，爷爷住在一楼的一个房间，我打小就和爷爷睡。

​ 我大概和爷爷睡到了小学二年级，能清晰记得的事情有几件。

​ 每天晚上爷爷会陪我在房间里看黑白电视，电视很小，颜色黄的发旧。夏天蚊子很多，我看见墙上有一只蚊子，我一巴掌就拍死了，白白的墙上立马红了一滩，我骄傲地喊道：“爷，你看，我拍的，这只蚊子好毒呀，吸了这么多血。”爷爷笑呵呵的，那滩血也一直留在了那面白墙上。

​ 一直短头发的我小时候有一次非要扎个小啾啾辫，结果晚上解开头发，躺在枕头上，头皮生疼，爷爷就温柔地给我揉揉。

​ 在外面，我拉粑粑，擦屁股没有纸，爷爷会用旁边的树叶给我擦。

​ 夏天我被蚊虫叮咬很痒，爷爷吐一口口水在手上，来回搓两下，擦在我起包的地方。长大了，我二十几岁了，傍晚吃完饭，腿被蚊虫叮的痒，爷爷看见后还是 会迅速用口水给我擦，是迅速，生怕被我反应过来，还偷笑。真的挺逗的，我还和我朋友说道，爷爷真的很可爱呀。

​ 幼儿园大班我考试考了双百，得了个文具盒，高高兴兴地往回走，爷爷就坐在马路边等我。看见我得奖，您也笑呵呵的。后来上一年级，我考试考砸了，被妈妈呵斥，您在吃饭也没有制止呢。爷爷是真有意思呀，宠着我也不惯着我。

​ 小时候我和你走很窄的夜路，我怕，想让你走前面，你非让我走前面，小时候不懂，现在长大才知道看见我在你才会安心；偶尔我想要择菜玩，但你也都不让我做，说会弄脏我的手；我喜欢吃很细很细的土豆丝，想自己切，结果你在旁边看的着急，事实证明，爷爷还是你刀工厉害呀，你切的土豆丝是我见过最细的……

​ 我喜欢吃辣条，爷爷会偷偷买给我吃，还让我吃完嘴巴要闭着不要嘘（缓解辣嘴巴做的动作），不然被妈妈发现会念叨。偶尔，还是会被妈妈发现，妈妈就会念叨着您不该给我买辣条，吃了上火，说您就溺爱，但您还是乐呵呵的，表面答应地好好好。

​ 只要我喜欢吃的，爷爷都会买给我。

​ 长大点后，我喜欢吃鸡爪鸡翅鸭爪，您都会给我买。虽然你也没有多少钱，但给我买吃的您绝对不含糊。

​ 每年夏天，您会去街上给我买小龙虾，洗刷的干干净净，妈妈炒给我们吃。长大后，和朋友吃饭，聊天无意中提到你啦，她说你爷爷是真的爱你呀，小时候大清早我看见你爷爷，问他干嘛去，你爷爷声音洪亮高兴地回应到：“我去街上给我帆子买虾子克。”爷爷，您的爱是有目共睹呀。

​ 每次放假写作业，下午你总会煮方便面或者粉丝给我吃，说是“摇钟”，有一种爱就是爷爷永远觉得你饿了没吃饱吧，然后每次吃完面的汤都是你喝掉，你说你不饿，煮好了给你吃你就是不吃，你说你只喜欢喝汤。以至于后来我直接煮好了面吃两口说不好吃我不吃了，你才吃，边吃还边说我心疼你，你乐呵呵的。后来长大了，方便面，糖果，果冻，藕粉都买给你吃，有的人会说吃了不健康对身体不好，但我们依旧买给你吃，就像小时候你买我吃那样，只要你喜欢你高兴。

爷爷，这样的一个人

​ 我从记事起，爷爷就是光头。其实爷爷的头发又黑又密，发质很硬，白发也不多。每次剃完头，不过两天就能摸到扎手的短茬，有点刺，却又带着一丝舒服的感觉。爷爷的头发生长得快，一个月就能长出一两厘米，又该去理发了。

​ 爷爷的额头上布满了深深的皱纹，认真时眉头紧锁，眼皮微微下垂，挡住了一些视线。高挺的鼻梁，鼻头像桃心一样；耳朵稍稍外翻；嘴角右上方有一颗显眼的黑痣，大家说是“口福痣”。也确实如此，爷爷这一生，吃喝不缺，尤其爱喝酒、吃辣，过得颇为滋润。年轻时的爷爷肯定是个俊朗的人，年老后也始终讲究仪表，端庄利落。

​ 他很爱干净，每天早晚梳洗，衣服勤换，穿得整整齐齐。房间收拾得一尘不染，床单、枕套也没有“老油味”，所以孩子们总喜欢在他床上玩。

​ 爷爷喜欢喝酒吃辣。听姑妈说，有一次他生病，医生叮嘱他不能再喝酒，他便戒了一阵。可那段时间人又瘦又没精神，后来实在受不了了，坚持要喝。姑妈在一旁哭，他还是喝了。奇怪的是，之后身体反倒慢慢恢复了。从此以后，他每餐都会喝上一小杯白酒，夏天还要加一瓶冰啤。饭桌上只要有辣椒，爷爷就吃得津津有味。

​ 爷爷也爱散步。每天天刚亮，就沿着村子转一圈，或者走到街上买点吃的。等我们醒来，他已经满载而归。如果没见到他，十有八九是去剃头了，剃完回来，总会带一袋吃的回来。

​ 他还爱聊天。下午没事就去邻居家串门，跟其他老人聊上一下午。有时候就在家门口的大树下摆几张椅子，三五老人围坐闲谈。因为爷爷耳朵有点背，大家说话都得提高音量。有时他没听清，就根据对方的嘴型或前文猜测意思。有时猜错了，闹出笑话，我们就哈哈大笑，再大声重复一遍。到了晚饭时，他下午的聊天就成了餐桌上的谈资。

​ 爷爷身体一直很好，八十几岁还能挑水干活。他常照料菜园，黄瓜、豇豆、辣椒，都是他的心血。虽然没得过大病，但骨折却有几次。冬天下雪路滑，他摔断了胳膊；我上高中那年，他被摩托车撞了，腿骨折。听到消息那一刻，我控制不住眼泪，见到他更是心疼。后来做了手术，腿里打了钢板，在家养了几个月就恢复了。

​ 爷爷也有点小娇气。身体哪里疼了，就会闹点脾气，不肯起床、不想吃饭，需要人哄。多叫几声，他便会慢慢配合。

照片拍摄于2022年1月31号 北京时间10:52

爷爷盼望的假期

​ 后来，我在武汉读书上班，小孩在市里上学，我们就只能在假期的时候回老家了。假期，也就成了爷爷的盼望。

2019年

​ 很遗憾，大学期间虽然已经有了手机，却几乎没有留下关于爷爷的影像。
​ 直到2019年研一暑假，那年姐姐没上班，在家带着小皓，我也回了老家，家里一下子热闹了起来。

​ 老家的院子宽敞明亮，承载了我们太多美好的回忆。

​ 正午时分，太阳火辣辣的，我们有时会在厢房的小廊下摆上桌椅吃饭；有时候干脆夹好饭菜，搬个小板凳坐在廊下就吃了，刚好避开毒辣的阳光。腿如果稍微往前伸一点，小腿马上就会被晒得生疼。

​ 夏天解暑少不了米茶，而每天煮米茶的任务就是爷爷的。
​ 他会在早上提前用电饭锅加水煮米茶米，等水烧开后再焖一会儿，等中午喝的时候就刚刚好。为了能更快凉下来，焖好后还会揭开盖子透透热气。

​ 说到这儿，我脑子里就浮现出爷爷煮米茶的画面：
​ 天气炎热，爷爷打着赤膊、穿着短裤，裤腰上挂着钥匙绳，头上戴着草帽，靠在灶台边，一边拿着勺子翻着米茶，一边看着锅里的动静。等盖好盖子，满意地拍拍手，从厨房走出去。

​ 晚上八点多天才黑。我们一般六点多快七点的时候吃饭，刚好太阳快落山了，余晖斜斜地洒在房顶的瓦片上。院子里不晒，又亮堂，还有阵阵微风，非常舒服。

​ 夏天是吃小龙虾的季节，爷爷会配点啤酒。每个人喝酒时都有自己独特的表情，爷爷也不例外。 照片里，他微微锁着眉头，嘴唇紧抿，伴随着咽下的动作，脖子上的筋线也若隐若现。看上去像是有点“难以下咽”，但细看之下，那其实是一种很享受的神情。

​ 夏天蚊虫多，爷爷总会在吃饭前提前点好蚊香，悄悄放在桌子底下；板凳旁还靠着一把扇子，吃饭时他时不时抬手帮我们驱蚊。就连电风扇的风向，也会特意调成朝我们这边吹。

​ 爷爷吃饭有他的一套：先吃几口米饭，喝点白酒，然后再来点啤酒。他爱吃辣，也爱吃小龙虾和田螺，但因为知道我们喜欢，他总是舍不得多吃。于是每次我们都一边给他夹菜，一边催着他快吃，爷爷也会笑着吃几口。

​ 他吃得可香了。龙虾壳吮得干干净净，田螺一挑一个准，牙签一转，一整颗就进了嘴里。我们总提醒他田螺后面的肠不能吃，爷爷却从不当回事，总说：“小时候哪有得挑啊，叶子都吃过呢！”
​ 听得我们哭笑不得，但爷爷吃得开心，我们也乐在其中。

​ 饭后，我们在院子里乘凉。屋旁的一小块地种满了树，微风吹来，树叶沙沙作响，蝉鸣不绝于耳，好像时间都慢了下来。这样的夜晚，是最温柔、最珍贵的回忆。

​ 那时的小皓才几个月大，姐姐刚在院子里给他洗完澡，水洒在地上，没一会儿就被热风晒干了。爷爷总是吃得快，吃完就抱起小皓哄着玩儿，让我们继续慢慢吃。

​ 小皓特别喜欢太爷，每次见到爷爷都会伸手要抱。爷爷会正正地坐在椅子上，左手托住小家伙的屁股，防着尿不湿掉下来，右手则穿过小胳膊，牢牢护着上身。小皓兴奋极了，抓着太爷的手就往嘴里啃，爷爷一定被啃得痒酥酥的，脸上全是止不住的笑意。

​ 那一幕，温柔、安稳、治愈，我悄悄打开了相机，把它记录了下来。

2020年

​ 2020年初，疫情突如其来，研二的寒假因此变得格外漫长。

​ 疫情爆发后，村口的路也封了，不能出门，但家里人多热闹，爷爷肯定也很开心。

​ 那段时间的记忆有些模糊，只记得一些零零散散的片段。

​ 冬天，老屋的一间屋子会生起农村特有的火炉。看似普通的“房”和“火炉”这两个词，其实都承载着二十几年的记忆。

​ 最初，那还不算一间“房”，更像是后院。老屋坐东朝西，一进门是三间厢房，从左到右依次是爷爷的卧室（后来为了方便行动，他搬到了厢房里）、用餐的厢房，以及厨房。紧挨着厨房的，就是那个熟悉的后院。

​ 小时候，后院里有一口井，墙上还留了个小洞，平时倒水就从那个小洞流出去。偶尔，院子里还会养几只鸡鸭。再往里走，就是厕所。

​ 后来，自来水普及了，不再需要井水，那口老井便被填了，小洞也封上了。慢慢地，后院加了顶，墙上开了窗，也装上了门，变成了一间真正的屋子。

​ 每到寒冬，我们就在这间“后院变身”的屋子里生起火炉，屋里暖洋洋的。爷爷有时也会在这里洗澡。

​ 火炉上面支着一张农村特有的铁桌子，吃饭的时候大家围坐在一起，既能用餐也能取暖。那会儿，爷爷的面前总会放一把椅子，小皓站在椅子上，爷爷双手紧握椅背，护着他的小身体，然后轻轻地前后晃动，就像在给他摇摇车。小皓玩得咯咯直笑，爷爷也跟着乐得合不拢嘴。

​ 冬天，我的脚冻伤了，时不时又痒又痛。中午吃过饭，我坐在院子里晒太阳，爷爷看见我一边挠脚一边皱眉头，走过来仔细看了看，说：“冻伤了。”

​ 没多会儿，他就从屋里找来药膏，蹲在我面前，示意我把脚抬起来。我把脚放在他腿上，爷爷开始给我擦药。他的手上布满了老茧，粗糙却温暖，揉在脚上像在轻轻挠痒，又特别舒服。

​ 一边擦着，爷爷一边叮嘱我：“这个要连着擦几天，别断了，才会好得快。”

​ 2022年8月20日，我们准备回荆门。小孩要开学了，我21号也要从荆门坐火车回武汉。爷爷一个人留在家，肯定又会孤单。姐姐打来电话说，不如把爷爷也带去荆门一起玩玩，我正有此意。于是，爸爸开车载着我们一行人：我、妈妈、骐骐、皓皓，还有爷爷，一起出发了。

​ 爷爷坐在副驾的位置，窗外的风景一路流转，也冲淡了往日每次离家时的低落。这次出行，格外温暖。

​ 这是爷爷第二次去荆门的新房。第一次是搬新家那天，爷爷和姑妈一起去参加了新居恭贺。那天爸妈想让爷爷留下过夜，爷爷却坚持要当天回去。

​ 而这一次，是我强行说服爷爷留下来多住几天的。他答应得意外地爽快，脸上也藏不住笑意。是啊，一个人待久了，终究会觉得孤单，他也渴望和儿孙、重孙们在一起，享受天伦之乐。

​ 到了荆门，妈妈特意做了一顿丰盛的晚饭。骐麒特别懂事，主动给太爷拿碗筷、盛饭。爷爷腰不好，茶几太矮，我就搬了个高凳放在沙发前，把饭菜、鸡汤，还有爷爷那一小杯酒都摆好，让爷爷坐在沙发上吃。吃饭时，爸妈不时提醒我给爷爷夹菜，爷爷一边笑着说：“吓人哦，吃不完了、不要夹了。”一边又吃得津津有味。

​ 最后，饭吃光了，酒也喝完了，爷爷的胃口比在胡集的时候好多了。我们笑着说，爷爷还是孤单了，看人多热闹了，心情好，胃口都跟着好了。

​ 饭后，妈妈在厨房洗碗，爸爸出门散步，我和爷爷、小孩们坐在沙发上看电视、玩耍。皓皓估计玩累了，靠在沙发上睡着了。骐麒在拼玩具，还时不时让太爷看看，爷爷看得非常认真。

​ 过了一会儿，我提议带爷爷出去逛逛。他常年住在镇上，很少见识小城市的模样。我特别希望爷爷能多看看外面的世界，因为他本身就是个爱走爱看、乐于尝新的老顽童。

​ 爷爷只带了根拐杖，没带扶椅，便靠着拐杖慢慢走着。骐麒一路贴心地搀着太爷。我们下楼去了超市，逛了一小会儿爷爷就有些累了，靠在货架上休息，我们便一同出来，坐在外面石阶上歇着。

​ 那天爷爷穿得整整齐齐：蓝色衬衫、黑色短裤，裤腰上还挂着常年不离身的钥匙，还有兵哥送给他的手表。骐麒穿着一件汉服小裙子。重孙对太爷的照顾，太爷眼里满是宠爱和满足。我忍不住拿出手机，记录下这温馨的一幕。本来是为了纪念，后来却成了怀念。

​ 休息了一会儿，我们便回家了。我给爷爷倒水、准备洗漱用品，怕他需要帮忙，我一直站在卫生间门口守着。

​ 我们商量着决定让爷爷睡主卧。骐麒把床铺整理得整整齐齐，贴心地放好水和纸巾，还把她最喜欢的娃娃摆在太爷枕边。我打开风扇，妈妈一边教爷爷怎么控制灯的开关，一边细细叮嘱：“半夜起床要慢慢来，小心点。”

​ 这也是我最后一次记录爷爷的画面。万万没想到，那一别，竟成永诀。再见之时，已是阴阳两隔。

​ 疫情放开后，爷爷不幸感染，短短几天，精神便开始恍惚。我当时还在武汉上班，远在他乡，并不知爷爷病情已然严重。直到2022年12月29日（腊月初七）周四下午四点，姐姐突然打来电话，说爷爷已经几天不吃不喝了，让我赶紧回家看看。

​ 那一刻，我的眼泪夺眶而出。但我仍未意识到问题的严重性，只以为爷爷一时不适，想着早点回去陪陪他，或许他看到我会开心些、恢复得快些。于是我请了假，收拾行李，甚至还不紧不慢地在路上给爷爷买了他爱吃的草莓和车厘子。

​ 路上，我一反常态地话特别多，一直在说爷爷的点滴。以为这一趟只是一次寻常的归家，谁知竟是诀别。

​ 后来我才真正明白，亲人的离去，不是一场突如其来的暴雨，而是一生漫长的潮湿。

​ 大家都说，爷爷活到这样的年纪，最后没遭太多罪，也算是喜丧了。但最后两年，还是吃了不少苦。有一次椅子没坐稳，不小心摔倒在地，导致盆骨粉碎性骨折。因为年纪太大了，恢复特别慢，那段时间爷爷吃了不少苦。

​ 爷爷最怕疼，疼得厉害时常常发脾气，有时候甚至会说些吓人的话。我们都知道，那是疼痛折磨下的情绪失控。好在后来慢慢修养了过来，身体也恢复了一些。

​ 爷爷走得安详，大家说是福气。

​ 我有时也会偷偷庆幸，也会深深地愧疚。

​ 庆幸的是，爷爷最后的两年，我已经参加工作，手里有点钱了。可以给您买吃的，买东西，时不时塞点钱给您。每次看您笑着接过东西、像个孩子一样开心，我的心里也特别踏实，特别满足。

​ 但也常常愧疚——等我终于能自己挣钱、能带您去看看外面的世界时，您却老了，腿脚不便了。我没能带您坐一次飞机，没能陪您去北京，去更远的地方看看。

​ 后来，我带爸妈去了北京。那次，我把您的照片放进了背包，带着它，一起走过那座城。我想，您或许也看见了。

​ 爷爷离开的头一年，我常常想起您就忍不住痛哭。每次在抖音上发和您有关的视频作品，我都在想，能不能把您在这个世界上留下的痕迹，再刻得深一点、久一点。

​ 后来，情绪慢慢平复了些，但想念依旧常在。我时常梦见您——梦里，您还活着，我无比庆幸；又或是梦见您离家出走，我到处寻找。那些梦总是让我在醒来时久久不能释怀。

​ 您的离去，不只带走了我深深的牵挂，也悄然改变了我看待世界的方式。我开始学着不再执拗、不再较劲。曾经存在隔阂的人，我也开始学着重新理解和接纳。那些曾经觉得无法释怀的事，慢慢都变得没那么重要了。

​ 我学着尊重自己，也尽量温柔地对待他人，保持开放、平和、豁达、积极的态度去生活。

​ 这是您留给我的最后一课，也是最深的一课。

​ 爷爷，我会一直记得您，一直爱您！！！

博客搭建

个人博客搭建教程

参考：b站视频、视频对应的博客文章

一、准备工具

下载Node.js、git、hexo(参考网上教程)

二、操作步骤

本地准备hexo项目 -> hexo配置文件中配置部署 -> 安装 Hexo Git 部署插件 -> github上创建仓库（仓库名：你的用户名.github.io）

typora写.md文件，文章开头写好 Front Matter ，文件保存在source/_posts下 —> 本地生成和预览博客（浏览器访问 http://localhost:4000 看效果）—> 部署上线到GitHub（静态文件推送到仓库的指定分支main）—> 几分钟后，访问线上博客（https://zoujinfan.github.io）

三、问题解决

问题：本地新增了一个文件，本地预览无问题，远程推送无更新。

解决：文件名是中文，导致了路径问题。将文件名改成了英文后重试成功。

四、背后原理介绍

  • Node.js是一个JavaScript运行环境，可以在 本地电脑/服务器 上运行 JavaScript应用程序

  • Hexo 是一个用Node.js编写的博客框架；Hexo 是一个Node.js应用，没有Node.js就无法启动Hexo命令（hexo init 都是Node.js脚本在跑）

    通俗理解：Hexo = Markdown + Node.js 魔法

    ​ Markdown学生写作文，只有在Node.js学校的环境下，才能参与高考Hexo

    • hexo init初始化项目结构，在空文件夹里自动创建hexo所需的完整目录和文件

    ​ 通俗理解：首次高考，一切初始化，环境准备、考试科目安排
